## ⚡ 快速启动（3步）

### 1️⃣ 安装依赖
```bash
# Windows用户双击运行
install.bat

# 或者使用命令
pip install -r requirements.txt
```

### 2️⃣ 配置数据库
打开 `backend/app.py`，修改第10-16行：
```python
DB_CONFIG = {
    'host': 'localhost',
    'user': 'root',
    'password': '你的密码',      # ← 改这里
    'database': '你的数据库名',   # ← 改这里
    'charset': 'utf8mb4'
}
```

### 3️⃣ 启动系统
```bash
# Windows用户双击运行
start.bat

# 或者使用命令
cd backend
python app.py
```

然后打开浏览器访问：**http://localhost:5000**

## ❓ 遇到问题？

### 问题1：数据库连接失败
➡️ 检查MySQL服务是否启动  
➡️ 确认用户名密码是否正确  
➡️ 运行 `python database_check.py` 检查

### 问题2：页面打不开
➡️ 确认后端服务已启动  
➡️ 检查5000端口是否被占用  
➡️ 查看后端控制台错误信息

### 问题3：数据不显示
➡️ 运行 `python database_check.py`  
➡️ 确认数据表存在且有数据  
➡️ 查看浏览器F12控制台

### 更多问题
查看 **USAGE.md** 的"常见问题"章节
